# 🤖 3D Animation Buddy

## 📌 Overview

3D Animation Buddy is an interactive virtual assistant project that simulates lifelike conversations through realistic 3D character animations, facial expressions, and body movements. The project focuses on creating an engaging and immersive user experience by combining animation with interactive communication.

## ✨ Features

- 🎭 Realistic 3D character animations
- 😊 Facial expression simulation
- 💬 Interactive chat interface
- 🕺 Smooth body movements
- 🎨 User-friendly interface
- ⚡ Real-time interaction

## 🛠️ Technologies Used

- Blender / Unity 
- Python / C# 
- HTML
- CSS
- JavaScript
